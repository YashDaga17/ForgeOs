# Security Policy

## Reporting a Vulnerability

We take the security of ForgeOS seriously. If you believe you have found a security vulnerability in this project, please do **NOT** report it publicly via a GitHub issue. Instead, report it privately to our security team.

Please email your report to `security@example.com` (replace with actual security inbox prior to publication). Include as much detail as possible, such as:
- Steps to reproduce the issue.
- Impact details.
- Sample exploit scripts or repository triggers.

We will acknowledge receipt within 48 hours and coordinate a patch within a reasonable timeline.

Thank you for helping keep our software secure!
